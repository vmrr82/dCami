Tabla de datos técnicos de vehículos y Tabla de deformaciones

El archivo a ejecutar es main.py.
Antes de ejecutarlo instalar dependencias con el siguiente comando en consola: pip install -r requirements.txt

Distribución de Carga: El botón de calcular, precisa de un valor en el campo de masa. Al modificar los valores del primer campo se obtienen el resto.

Centro de gravedad: Realiza la suma del campo Vol. Ant y Centro de Gravedad (Desde Eje Ant).
